¿Que es el DOM?


El DOM (Modelo de Objetos del Documento) es una interfaz de programación que representa un documento HTML o XML como una estructura de árbol de objetos.

¿Que es el DOM de animación?


Es el que se encarga de las transiciones entre estados, de modo que es posible cambiar entre dos o más estados lo que crea la animacion, permite que los elementos se muevan, cambien de tamallo, color, opacidad, entre otras.

¿Que es el DOM de eventos?


Acciones que ocurren en la pagina web, con clics del mouse, pulsaciones de teclas o cambios en campos de formulario.

¿Que es el DOM de CSS?


Es un conjunto de APIs que permite manipular CSS desde JavaScript. Permite leer y modificar el estilo de CSS de forma dinámica. Dándole estilos y estética al contenido hecho en el DOM. Transforma los estilos en objetos que son entendibles para el navegador, dando estilos al DOM.​

Color animation


La animación de color CSS implica una transición suave de las propiedades de color de un elemento durante un tiempo determinado. Esto puede aplicarse al color de fondo, al color (para texto), al color del borde y a otras propiedades relacionadas con el color.

Delay


Permite establecer cuánto tiempo debe esperar una animación antes de comenzar. Básicamente, funciona como un temporizador que retrasa el inicio de la transición después de que el usuario realiza una acción, como pasar el cursor sobre un elemento o hacer clic en él. Este tiempo se puede definir en segundos (s) o milisegundos (ms), dependiendo de la precisión que necesites.

Animation-timing-function


La propiedad establece cómo progresa una animación a través de la duración de cada ciclo. Define la curva de velocidad de una animación y controla cómo progresa la animación a lo largo de su duración.

Animation-direction


indica si la animación debe retroceder hasta el fotograma de inicio al finalizar la secuencia o si debe comenzar desde el principio al llegar al final.

Animation-fill-mode


Determina cómo se aplican los estilos de un elemento antes y después de que una animación comience y termine. Por defecto, las animaciones CSS solo afectan a un elemento durante su tiempo de ejecución activa, lo que significa que no aplican estilos durante ningún retraso de animación ni después de que esta finalice.
